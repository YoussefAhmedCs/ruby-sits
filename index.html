Learning Ruby as a Beginner

I created this website while exploring Ruby and learning more about web development. My goal was to build a simple website that introduces Ruby in an easy way.

Ruby caught my attention because its syntax is clean and easy to understand. Compared to some languages I have used before, Ruby feels more natural to read.

Why I Chose Ruby

One of the things I like about Ruby is that it helps beginners focus on programming concepts without getting distracted by complicated syntax.

Ruby is also used with Ruby on Rails, a framework that helps developers build web applications faster.

Resources I Used

  • Ruby Official Quickstart Guide
  • Ruby on Rails Getting Started Guide
  • YouTube tutorials and practice exercises
  • Reading sample Ruby programs and experimenting with code
