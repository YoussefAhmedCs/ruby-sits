My First Steps with Ruby

When I started learning Ruby, I focused on the basics such as variables, loops, and methods. These concepts helped me understand how Ruby programs work.

One thing I liked about Ruby is that the syntax is easy to read, which makes learning more enjoyable for beginners.

Topics I Explored

  • Variables and data types
  • Conditional statements
  • Loops and iteration
  • Methods and functions
  • Basic object-oriented programming

Simple Ruby Example

name = "Youssef"

puts "Hello, #{name}"

3.times do
  puts "Learning Ruby"
end

This example prints a greeting and then displays a message three times using a loop.
